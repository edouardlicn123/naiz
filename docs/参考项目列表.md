# 参考项目列表

| 项目 | 说明 |
|------|------|
| MHVNVisualNovelEngine | PC-98 视觉小说引擎，数据驱动，支持脚本汇编、图片打包、链接等工具链 |
| master.lib | PC-98 综合库（图形/声音/I/O），16 位 x86 汇编，为 DOS 游戏提供硬件抽象层 |
| ReC98 | 东方 Project PC-98 五作（TH01-TH05）的逆向还原工程，目标实现位元完美编译 |
| pmdmini | PC-98 上的 PMD/MDX 音乐播放库，模拟 YM2203/YM2608 FM 合成芯片 |
| xsystem35-sdl2 | AliceSoft System 3.x 游戏引擎的跨平台移植，支持 Rance 等游戏 |
| xsys35c | AliceSoft System 3.x 游戏引擎的编译/反编译工具链 |
| djlsr | DJGPP，MS-DOS 下的 32 位保护模式 C/C++ 开发环境（GCC 移植） |
| gdc_test | 在 Neko Project 21/W 上测试 PC-98 GDC 图形输出的实验代码 |
| ps2busmouse98 | 将 PS/2 鼠标转换为 PC-9801 总线鼠标的 Arduino 转换器 |
| np21w | PC-9800 系列模拟器 Neko Project 21/W，支持后期 PC-98 硬件 |
| 98imgtools | PC-98 模拟器磁盘映像（HDD/HDI/NHD/SLH）读写工具集 |
| 98fmplayer | PC-98 FM 驱动仿真器，支持 PMD/FMP 格式回放 |
