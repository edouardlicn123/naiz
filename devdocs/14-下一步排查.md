1. 查询 NP2kai 对配置文件键名的支持
   - run `strings sdlnp21kai_sdl2 | grep -iE 'bios|hdd|boot|disk|drive' | sort -u`
   - 将输出记录

2. 备份当前配置
   - `cp core/sdlnp21kai/np21kai.cfg core/sdlnp21kai/np21kai.cfg.bak`

3. 创建最小测试配置（仅 BIOS 与 HDI）
   ```ini
   [NP2]
   BIOS_PATH = /home/edo/naiz/tools/pc98bios/ROM.ROM
   HDD1FILE   = /home/edo/naiz/tools/DOS62.hdi
   BOOT_DRIVE = HDD
   ```
   将其写入 `core/sdlnp21kai/np21kai.cfg`

4. 运行测试并记录日志
   - `cd /home/edo/naiz && ./testtemp.sh > logs/debug.log 2>&1`
   - 检查 `logs/debug.log` 是否出现 DOS 提示符 `A:\>`。

5. 启用 NP2kai 详细日志（如可)**
   - `sdlnp21kai_sdl2 --config core/sdlnp21kai/np21kai.cfg -V > logs/verbose.log 2>&1`

6. 验证 BIOS ROM
   - `file tools/pc98bios/ROM.ROM`
   - `hexdump -C tools/pc98bios/ROM.ROM | head`

7. 验证 HDI 内容
   - `python3 - <<'PY'
import os
path='tools/DOS62.hdi'
print('size', os.path.getsize(path))
print(open(path, 'rb').read(512)[:64])
PY`

8. 如仍失败，使用 QEMU 测试
   - `qemu-system-i386 -boot c -hda tools/DOS62.hdi -L tools/pc98bios`

9. 记录最终可行的 `np21kai.cfg` 与日志。
