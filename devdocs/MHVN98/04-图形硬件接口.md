# 图形硬件接口

MHVN98 直接操作 PC-98 的三个图形加速芯片：**EGC** (增强图形控制器)、**GRCG** (图形加速器) 和 **GDC** (图形显示控制器)。

## VRAM 内存布局

PC-98 在 640×400 16 色模式下使用 4 位平面 (bit-plane) 显存布局：

```
平面 0 (段 0xA800): 位 0
平面 1 (段 0xA900): 位 1
平面 2 (段 0xAA00): 位 2
平面 3 (段 0xAB00): 位 3
```

每行 80 字节（640 像素 ÷ 8 位/字节），400 行，每平面 32000 字节。

### GDC (图形显示控制器) — `pc98_gdc.c/h`

GDC 是 PC-98 显示控制的核心。所有操作通过 I/O 端口 `0x6A` (参数) 和 `0x6C` (命令) 进行。

**关键函数**：

```c
void GDCSetDisplayMode(unsigned short htotal, unsigned short vtotal, unsigned short hsync);
// 设置显示模式，含安全范围检查

void GDCSetMode1(unsigned short mode);
// 模式 1: LINEDOUBLE_ON(0x4000) | COLOUR(0x2000)

void GDCSetMode2(unsigned short mode);
// 模式 2: GDC_MODE2_16COLOURS = 0x0004

void GDCStopText(void);     // 关闭文本层
void GDCStartGraphics(void); // 开启图形层
void GDCStartText(void);     // 开启文本层
void GDCStopGraphics(void); // 关闭图形层

void GDCSetPaletteColour(unsigned char index, unsigned short r, unsigned short g, unsigned short b);
// 设置调色板（4bpc 值：0x0-0xF）

void GDCScrollSimpleGraphics(unsigned short offset);
// 简单垂直滚动（画面震动效果用）

void GDCSetGraphicsLineScale(unsigned char scale);
// 行缩放
```

**绘图函数**：
```c
void GDCDrawLine(short x1, short y1, short x2, short y2);
void GDCStrokeRectangle(short x, short y, short w, short h);
void GDCFillRectangle(short x, short y, short w, short h);
void GDCDrawArc(short x, short y, short r, short startAngle, short endAngle);  // 桩
```

GDC 内联函数 (`pc98_gdc.h`) 提供了芯片寄存器的直接 I/O 访问：
```c
static inline void GDCWriteParam(unsigned short param);  // 端口 0x6A
static inline void GDCWriteCommand(unsigned short cmd);  // 端口 0x6C
static inline unsigned short GDCReadStatus(void);        // 端口 0x6A
```

### EGC (增强图形控制器) — `pc98_egc.c/h`

EGC 是 PC-98 后期机型提供的图形加速硬件，支持高速位块传输和逻辑运算。MHVN98 强制要求此硬件。

**关键函数**：

```c
void EGCEnable(void);
void EGCDisable(void);

void EGCClearScreen(void);
void EGCSetToMonochromeDrawMode(void);
void EGCSetVRAMBlitMode(void);
void EGCSetFGColour(unsigned char col);
void EGCSetBGColour(unsigned char col);
```

**EGC 寄存器配置宏** (`pc98_egc.h`)：

```c
// 端口 0x4A0-0x4AF
void EGCSetPlaneAccess(unsigned char mask);  // 启用/禁用各平面写入
void EGCSetPatternAndReadSource(unsigned char src);
void EGCSetReadWriteMode(unsigned char mode);
void EGCSetMask(unsigned short mask);
void EGCSetFGColour(unsigned short col);
void EGCSetBGColour(unsigned short col);
void EGCSetBitAddressTransferDirection(unsigned short mode);  // 含位地址偏移
void EGCSetBitLength(unsigned short length);
```

**EGC 读写模式枚举**：
```c
#define EGC_SOURCE_CPU          0x0000
#define EGC_SOURCE_PATTERN      0x0020
#define EGC_WRITE_ROPSHIFT      0x000E  // 通常与 SOURCE_CPU 组合
#define EGC_ROP_SRC             0x00   // 直接复制源
#define EGC_ROP_SRC_AND_DST     0x80
#define EGC_ROP_SRC_OR_DST      0xF0
#define EGC_ROP_SRC_XOR_DST     0x60
#define EGC_ROP_DST             0xAA
```

### GRCG (图形加速器) — `pc98_grcg.c/h`

GRCG 是比 EGC 更基础的图形加速硬件，主要用于快速填充操作。MHVN98 使用了其 TDW (Tile Data Write) 模式。

**关键函数**：

```c
void GRCGClearScreen(void);         // 快速清屏
void GRCGClearLine(short y);        // 快速清行
void GRCGEnable(void);
void GRCGDisable(void);
```

**GRCG 寄存器宏** (`pc98_grcg.h`)：

```c
// 端口 0x7C-0x7F
void GRCGSetWriteMode(unsigned char mode);
// TDW 模式: 0x80
// RMW 模式: 0x00

void GRCGSetTileRegisters(unsigned short tile0, unsigned short tile1,
                          unsigned short tile2, unsigned short tile3);
void GRCGSetColourFill(unsigned char col);
```

## 硬件关系

```
GDC (I/O 0x6A/0x6C)          │ 控制显示时序、模式、滚动、绘图
   └── 设置 VRAM 段: 0xA800-0xAB00

EGC (I/O 0x4A0-0x4AF)        │ 高速位块传输、逻辑运算
   └── 加速字体渲染、图像绘制
       └── 逐行写入时自动展开位面

GRCG (I/O 0x7C-0x7F)         │ 快速填充
   └── 清屏、清行
```

## 中断复位

```c
void GDCInterruptReset(void);
```
在设置显示模式后必须调用，清除 GDC 的中断请求状态，避免主循环立即触发不必要的 VSync。
