# 平台抽象层

**路径**: `src/platform/`

MHVN98 的平台层封装了 MS-DOS 和 PC-98 硬件相关的操作。所有功能通过 IA-16 实模式下的内联汇编或 BIOS/DOS 中断实现。

## 文件 I/O — `filehandling.c/h`

基于 DOS 系统调用 (`INT 21h`) 的文件操作包装：

| 函数 | DOS 调用 | 功能 |
|------|----------|------|
| `OpenFile` | AH=3Dh | 打开文件（只读/写入） |
| `CreateFile` | AH=3Ch | 创建新文件 |
| `CloseFile` | AH=3Eh | 关闭文件 |
| `ReadFile` | AH=3Fh | 从文件读取 |
| `WriteFile` | AH=40h | 写入文件 |
| `SeekFile` | AH=42h | 移动文件指针 |
| `DeleteFile` | AH=41h | 删除文件 |

`fileptr` 类型为 `unsigned short`（DOS 文件句柄）。

```c
int OpenFile(const char* path, unsigned char mode, fileptr* handle);
int ReadFile(fileptr handle, unsigned short count, __far unsigned char* buf, unsigned int* bytesRead);
```

## 内存分配 — `memalloc.c/h`

基于 DOS 系统调用 (`INT 21h`) AH=48h/49h/4Ah：

```c
void* MemAlloc(unsigned short paras);     // 分配（单位：16 字节段落）
void MemFree(void* block);
void* MemRealloc(void* block, unsigned short newParas);
```

## 端口 I/O — `x86ports.h`

提供类型安全的端口读写内联函数：

```c
static inline void PortOutB(unsigned short port, unsigned char val);
static inline unsigned char PortInB(unsigned short port);
// 支持 8/16/32 位、标准端口 (0x00-0xFF) 和扩展端口 (0x100-0xFFFF)
```

## 段寄存器操作 — `x86segments.h`

实模式段寄存器管理：

```c
unsigned short GetDS(void);
void SetES(unsigned short seg);
void SetFS(unsigned short seg);
// ... 以及跨段内存操作的宏
```

关键宏定义：
- `GDC_PLANES_SEGMENT` = `0xA800`（VRAM 平面段地址）
- 四个显存平面的段地址：`0xA800` + 平面号 × `0x100`

## x86 字符串操作 — `x86strops.h`

手动优化的内存复制函数，避免依赖标准库：

```c
void Memcpy16Near(const void* src, void* dst, unsigned short words);
void Memset16Near(unsigned short val, void* dst, unsigned short words);
// 支持 Near (同段) 和 Far (跨段) 版本
```

## x86 中断管理 — `x86interrupt.h`

```c
#define INTFUNC __attribute__((__interrupt__))
void swint(unsigned char vector);       // 触发软件中断
InterruptFuncPtr GetInterruptFunctionRaw(unsigned char vec);  // 读取 IVT
void SetInterruptFunctionRaw(unsigned char vec, InterruptFuncPtr func);  // 设置 IVT
void intsoff(void);  // CLI
void intson(void);   // STI
```

## DOS 系统调用 — `doscalls.h`

大批 DOS `INT 21h` 功能的 C 封装，包括：
- `DOSConsoleWriteString` — 控制台输出
- `DOSReadFile` / `DOSWriteFile`
- `DOSGetSetInterruptVector`
- DOS 内存块控制功能

## PC-98 中断控制器 — `pc98_interrupt.h`

8259A PIC（可编程中断控制器）的管理：

```c
// 主片端口: 0x00 (IMR), 0x02 (OCW1/2/3)
// 从片端口: 0x04 (IMR), 0x06 (OCW1/2/3)
unsigned char PC98GetPrimaryInterruptMask(void);
void PC98SetPrimaryInterruptMask(unsigned char mask);
void PC98AddPrimaryInterrupts(unsigned char mask);
void PC98RemovePrimaryInterrupts(unsigned char mask);
void PC98PrimaryEndOfInterrupt(void);
// 从中断控制器同理
```

中断掩码定义：
```c
#define INTERRUPT_MASK_VSYNC   0x20
#define INTERRUPT_MASK_TIMER   0x01
#define INTERRUPT_MASK_KEYBOARD 0x02
```

中断向量常量：
```c
#define INTERRUPT_VECTOR_VSYNC   0x08  // VSync 中断向量号
#define INTERRUPT_VECTOR_TIMER   0x1C
#define INTERRUPT_VECTOR_KEYBOARD 0x09
```

## CRT BIOS — `pc98_crtbios.h`

通过 `INT 18h` 操作 PC-98 的 CRT BIOS：

```c
void CRTBIOSSetGraphicMode(unsigned char mode);  // 设置图形模式
void CRTBIOSSetDisplayPage(unsigned int page);    // 设置显示页
void CRTBIOSSetGraphicPalette(void);              // 设置调色板
// 文本/图形层切换等
```
