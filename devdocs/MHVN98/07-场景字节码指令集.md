# 场景字节码指令集

**源文件**: `src/sceneengine.c:568-1195`（`SceneDataProcess` 的 `switch` 语句）

所有指令编码为单字节操作码 + 变长参数。场景数据经过 LZ4 压缩存储在 `SCENE.DAT` 中。

## 流程控制

| 操作码 | 指令 | 参数 | 行为 |
|--------|------|------|------|
| `0x00` | `gotoscene` | `[uint16 scene_num]` | 跳转到指定场景；`0xFFFF` = 故事结束 |
| `0x01` | `jmp` | `[int16 offset]` | 无条件相对跳转 |
| `0x02` | `jz/je` | `[int16 offset]` | Z 标志为 1 时跳转 |
| `0x03` | `jnz/jne` | `[int16 offset]` | Z 标志为 0 时跳转 |
| `0x04` | `jn/jl` | `[int16 offset]` | N 标志为 1 时跳转 |
| `0x05` | `jp/jge` | `[int16 offset]` | N 标志为 0 时跳转 |
| `0x06` | `jle` | `[int16 offset]` | N 或 Z 标志为 1 时跳转 |
| `0x07` | `jg` | `[int16 offset]` | N 和 Z 标志均为 0 时跳转 |

## 调色板操作

| 操作码 | 指令 | 参数 | 行为 |
|--------|------|------|------|
| `0x08` | `palsetcol` | `[uint5 r, uint5 g, uint5 b]` (2 字节打包) | 设置混合调色板为纯色 |
| `0x09` | `paladdcol` | `[u5 r, u5 g, u5 b]` (2 字节打包) | 混合调色板 = 主调色板 + 固定颜色 |
| `0x0A` | `palsetlum` | `[int8 mod]` | 亮度修改 (-128~+127) |
| `0x0B` | `palsetsat` | `[uint8 mod]` | 饱和度修改 (×mod/128) |
| `0x0C` | `palsethue` | `[int8 mod]` | 色调旋转 (±mod/128 × 180°) |
| `0x0D` | `palcolourise` | `[u5 r, u5 g, u5 b]` (2 字节打包) | 着色：所有颜色使用指定色度 |
| `0x0E` | `palinvert` | 无 | 反色 |

## 文本指令

| 操作码 | 指令 | 参数 | 行为 |
|--------|------|------|------|
| `0x10` | `nexttext` | 无 | 显示下一条文本（自动生成） |
| `0x11` | `text` | `[uint16 textnum]` | 显示指定文本，等待用户输入 |
| `0x12` | `charname` | `[uint16 charnum]` | 设置当前角色名；`0xFFFF` = 清除 |
| `0x13` | `deltext` | 无 | 擦除文本框内容 |

`text` 自动合并为连续的 `nexttext`（汇编器行为）。

## 选择指令

| 操作码 | 指令 | 参数 | 行为 |
|--------|------|------|------|
| `0x14` | `ynchoice` | 无 | 是/否选择；结果存入 Z 标志（1=是） |
| `0x15` | `choice2` | `[uint16 var, u16 text1, u16 text2]` | 2 个选项；选中序号写入 var |
| `0x16` | `choice3` | `[u16 var, u16 text1, u16 text2, u16 text3]` | 3 个选项 |
| `0x17` | `choice4` | `[u16 var, u16 text1..text4]` | 4 个选项 |

选择指令在运行时暂停 VM，等待用户通过 ↑/↓/Enter 操作选择。

## 异步效果指令

| 操作码 | 指令 | 参数 | 行为 |
|--------|------|------|------|
| `0x18` | `bfadein` | `[u3 amt, u5 speed]` | 从黑色淡入 |
| `0x19` | `bfadeout` | `[u3 amt, u5 speed]` | 淡出至黑色 |
| `0x1A` | `wfadein` | `[u3 amt, u5 speed]` | 从白色淡入 |
| `0x1B` | `wfadeout` | `[u3 amt, u5 speed]` | 淡出至白色 |
| `0x1C` | `pfadein` | `[u3 amt, u5 speed]` | 调色板混合淡入（→混合调色板） |
| `0x1D` | `pfadeout` | `[u3 amt, u5 speed]` | 调色板混合淡出（→主调色板） |
| `0x1E` | `phuerotate` | `[uint8 speed]` | 色调连续旋转；speed=0 停止 |
| `0x1F` | `shake` | `[u6 amp, u5 period, u5 damp]` | 垂直画面震动 |
| `0x0F` | `nowait` | 无 | 修饰符：紧跟的指令不阻塞 VM |

## 变量/算术指令

| 操作码 | 指令 | 参数 | 行为 |
|--------|------|------|------|
| `0x20` | `lut2` | `[u16 var, i16 v0, i16 v1]` | 以 var 为索引的 2 元素查表 |
| `0x21` | `lut3` | `[u16 var, i16 v0..v2]` | 3 元素查表 |
| `0x22` | `lut4` | `[u16 var, i16 v0..v3]` | 4 元素查表 |
| `0x23` | `swapzn` | 无 | 交换 Z 和 N 标志 |
| `0x24` | `setvi` | `[u16 var, i16 imm]` | var = imm |
| `0x25` | `setvv` | `[u16 var1, u16 var2]` | var1 = var2 |
| `0x26` | `csetvi` | `[u16 var, i16 imm]` | Z=1 时 var=imm |
| `0x27` | `csetvv` | `[u16 var1, u16 var2]` | Z=1 时 var1=var2 |
| `0x28` | `cmpvi` | `[u16 var, i16 imm]` | 比较 var 与 imm |
| `0x29` | `cmpvv` | `[u16 var1, u16 var2]` | 比较 var1 与 var2 |
| `0x2A` | `addvi` | `[u16 var, i16 imm]` | var += imm |
| `0x2B` | `addvv` | `[u16 var1, u16 var2]` | var1 += var2 |
| `0x2C` | `subvi` | `[u16 var, i16 imm]` | var -= imm |
| `0x2D` | `subvv` | `[u16 var1, u16 var2]` | var1 -= var2 |

### 标志操作

| 操作码 | 指令 | 参数 | 行为 |
|--------|------|------|------|
| `0x2E` | `ldflg` | `[u16 flag]` | 读取标志 → Z 标志 |
| `0x2F` | `stflg` | `[u16 flag]` | Z 标志 → 写入标志 |

### 图像/精灵（部分为桩）

| 操作码 | 指令 | 参数 | 状态 |
|--------|------|------|------|
| `0x30` | `setbg` | `[uint16 bg]` | ✅ 加载并显示背景 |
| `0x31` | `addbgvar` | `[uint16 var]` | ❌ 桩 |
| `0x32` | `subbgvar` | `[uint16 var]` | ❌ 桩 |
| `0x34` | `setspr0` | `[uint16 spr]` | ❌ 桩 |
| `0x35` | `addspr0var` | `[uint16 var]` | ❌ 桩 |
| `0x36` | `subspr0var` | `[uint16 var]` | ❌ 桩 |
| `0x38` | `setspr1` | `[uint16 spr]` | ❌ 桩 |
| `0x39` | `addspr1var` | `[uint16 var]` | ❌ 桩 |
| `0x3A` | `subspr1var` | `[uint16 var]` | ❌ 桩 |
| `0x3C` | `setspr2` | `[uint16 spr]` | ❌ 桩 |
| `0x3D` | `addspr2var` | `[uint16 var]` | ❌ 桩 |
| `0x3E` | `subspr2var` | `[uint16 var]` | ❌ 桩 |

操作码 `0x33`、`0x37`、`0x3B`、`0x3F` 保留给未来的背景/精灵扩展。
操作码 `0x40-0xFF` 未分配。
