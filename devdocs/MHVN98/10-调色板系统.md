# 调色板系统

**源文件**: `src/palette.c` (482 行), `src/palette.h` (67 行)

## 三层调色板架构

MHVN98 使用三级调色板系统，各司其职：

```
背景/精灵数据 → mainPalette (主调色板)
                       │
             经效果指令修改 → mixPalette (混合调色板)
                                │
              main + mix 混合 → outPalette (输出调色板)
                                    │
                       亮度/色调调整 → GDC 硬件调色板寄存器
```

### 数据结构

```c
ColourRGB mainPalette[16];  // 基础调色板，从背景图像加载
ColourRGB mixPalette[16];   // 混合调色板，用于效果叠加
ColourRGB outPalette[16];   // 输出调色板，最终送入硬件

typedef struct {
    unsigned char r;  // 8bpc (0x00-0xFF)
    unsigned char g;
    unsigned char b;
} ColourRGB;
```

## YUV 色彩空间运算

引擎内部使用 YUV 色彩空间进行调色板变换。

### 转换公式

```c
typedef struct {
    short y;  // 2.14 定点数
    short u;
    short v;
} ColourYUV;
```

**RGB → YUV** (4.28 中间精度):
```
Y =  3483 * R + 11718 * G +  1183 * B
U = -1637 * R -  5507 * G +  7144 * B
V = 10076 * R -  9152 * G -   924 * B
```

**YUV → RGB** (4.28 中间精度):
```
R = Y               + 20977 * V
G = Y -  3520 * U -  6236 * V
B = Y + 34865 * U
```

所有结果经过四舍五入和 `0x00-0xFF` 钳位。

### 查找表

```c
const short c8bpcto2p14pcTable[256];      // 8bpc → 2.14 定点 (×64)
const unsigned char c5bpcto8bpcTable[32]; // 5bpc → 8bpc (×~8.23)
```

`c5bpcto2p14pcTable` 将 8 位颜色值转换为 2.14 定点格式，避免运行时除法。`c5bpcto8bpcTable` 用于将从操作码 `0x08`/`0x09`/`0x0D` 接收的 5bpc 值扩展为 8bpc。

## 调色板效果实现

### 1. 纯色设置 (`SetMixPaletteToSingleColour`/`SetMixPaletteToSingleColour5bpc`)

将所有 16 个混合调色板条目设为相同的 RGB 值。用于 `palsetcol` 指令。

### 2. 颜色加法 (`SetMixPaletteToMainAdd`/`SetMixPaletteToMainAdd5bpc`)

```
mixPalette[i] = clamp(mainPalette[i] + (r, g, b))
```

用于 `paladdcol` 指令。5bpc 版本支持正负值（最高位为符号位）。

### 3. 亮度修改 (`SetMixPaletteToMainLuminosityMod`)

```
亮度降低 (mod < 0): mix = main × (16384 + mod) / 16384
亮度提高 (mod > 0): mix = white - (white - main) × (16384 - mod) / 16384
```

`mod` 是 2.14 定点数，范围 `[-16384, 16384]`。完整的亮度变化路径：
```
白 (mod=+16384) ← main ← 黑 (mod=-16384)
```

### 4. 饱和度修改 (`SetMixPaletteToMainSaturationMod`)

```
mixColor[i] = YUVtoRGB(Y(main[i]), U(main[i]) × mod/4096, V(main[i]) × mod/4096)
```

`mod` 是 4.12 定点数，0 = 灰阶, 4096 = 不变。

### 5. 色调旋转 (`SetMixPaletteToMainHueMod`)

```
hue = atan2(V(main[i]), U(main[i]))
hue += mod
newU = sat × cos(hue)
newV = sat × sin(hue)
mix = YUVtoRGB(Y(main[i]), newU, newV)
```

`mod` 是 1.15 定点数，单位是整周旋转。`0xFFFF` ≈ 360° 旋转，`0x8000` ≈ 180°。

### 6. 着色 (`SetMixPaletteToMainColourised`)

```
mix[i] = YUVtoRGB(Y(main[i]), U(目标色), V(目标色))
```

保留亮度，替换色度，使所有颜色偏向指定色调。

### 7. 反色 (`SetMixPaletteToMainInvert`)

```
mix[i].r = 0xFF - main[i].r
mix[i].g = 0xFF - main[i].g
mix[i].b = 0xFF - main[i].b
```

### 8. 调色板混合 (`MixPalettes`)

```c
outPalette[i] = lerp(mainPalette[i], mixPalette[i], mixAmt/255)
```

0 = 纯 main, 255 = 纯 mix。

## 显示输出

### `SetDisplayPaletteToOut()` (行 408–422)

将 outPalette 写入 GDC 硬件调色板寄存器：
```
GDC 值 = (8bpc_value + 0x08) / 0x11  → 转为 4bpc (0x0-0xF)
```

### `SetDisplayPaletteToOutBrightnessModify()` (行 424–441)

在写入前添加亮度偏移（用于黑白淡入淡出）：
```
GDC 值 = clamp((8bpc + 0x08 + brightness_add) / 0x11, 0, 0xF)
```

### `SetDisplayPaletteToOutHueRotate()` (行 443–475)

在写入前对每个颜色进行 YUV 色调旋转，用于 `phuerotate` 指令的连续色调旋转效果。
