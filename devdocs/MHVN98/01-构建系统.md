# 构建系统

## 工具链

MHVN98 使用 **gcc-ia16** (`ia16-elf-gcc`) 交叉编译器，将 C 源码编译为 i386 兼容的 MS-DOS MZ `.EXE`。

- **架构**: `i386`
- **内存模型**: `small`（代码段 64KB，数据段 64KB）
- **输出格式**: `elf_i386_msdos_mz`

## Makefile 分析

### 目录布局

```
MHVN98/
├── Makefile
├── wholeprog.c           # 全程序优化入口（#include 所有 .c 文件）
├── src/
│   ├── *.c, *.h, *.asm   # 核心源码
│   └── platform/         # 平台层源码
├── obj/                   # 编译中间文件
├── bin/                   # 输出目录
└── genasm/               # 汇编列表（调试用）
```

### 编译变量

```makefile
CFLAGSBASE    := -march=i386 -mcmodel=small
CFLAGSRELEASE := $(CFLAGSBASE) -O2
LINKLIBS      := -li86  # ia16 运行时库
LINKFLAGSBASE := -Wl,-melf_i386_msdos_mz,-M,-static,--relax,-n,--no-dynamic-linker,--print-memory-usage
```

### Make 目标

| 命令 | 说明 |
|------|------|
| `make` / `make default` | 发布构建，`-O2` 优化 |
| `make debug` | 调试构建（`-O2`，但不同链接方式） |
| `make showasm` | 生成汇编列表文件到 `genasm/` |
| `make clean` | 清理 `obj/` 和 `bin/` |

### 两种构建模式

**模式 A — 全程序优化**（默认）：
```makefile
default : $(BUILD) $(TARGET) wholeprog.o $(OFILESASM)
	ia16-elf-gcc $(CFLAGS) $(LINKFLAGS) -o $(TARGET)/MHVN98.EXE wholeprog.o $(OFILESASM) $(LINKLIBS)
```
`wholeprog.o` 由 `wholeprog.c` 编译而来，后者 `#include` 所有 `.c` 文件：
```c
#include "src/platform/filehandling.c"
#include "src/platform/memalloc.c"
#include "src/platform/pc98_chargen.c"
#include "src/platform/pc98_egc.c"
#include "src/platform/pc98_grcg.c"
#include "src/platform/pc98_gdc.c"
#include "src/platform/pc98_keyboard.c"
#include "src/main.c"
#include "src/lz4.c"
#include "src/unicode.c"
#include "src/palette.c"
#include "src/stdbuffer.c"
#include "src/rootinfo.c"
#include "src/gpimage.c"
#include "src/graphics.c"
#include "src/sceneengine.c"
#include "src/textengine.c"
```
这为编译器提供最大内联和优化空间（`-fwhole-program`）。

**模式 B — 独立编译**（debug）：
```makefile
$(TARGET)/MHVN98.EXE : $(OFILES)
	ia16-elf-gcc $(CFLAGS) $(LINKFLAGS) -o $@ $^ $(LINKLIBS)
```
逐个编译 `.c` → `.o`，然后链接。

### 汇编文件处理

```makefile
$(BUILD)/%.o : $(MAINSRC)/%.asm
	ia16-elf-as $< -o $@ -O2
```

两个汇编文件：
- `isr.asm` — VSync 中断服务例程
- `lz48086.asm` — LZ4 解压缩算法（8086 汇编实现）

### 编译流程

```
                     wholeprog.c
                          │
                    #include 所有 .c
                          │
                    ia16-elf-gcc -fwhole-program
                          │
                    wholeprog.o ——┐
                    isr.asm      │
                    lz48086.asm  │
                          │      │
                    ia16-elf-as  │
                          │      │
                    *.o ─────────┘
                          │
                    ia16-elf-gcc -Wl,-melf_i386_msdos_mz
                          │
                    bin/MHVN98.EXE
```

## 运行需求

- PC-98 实模式 DOS 环境
- EGC（Enhanced Graphics Charger）硬件支持
- PMD（如果使用音乐功能）
- `ROOTINFO.DAT` 必须在当前目录
- 字体文件 + 场景 + 文本数据文件按 `ROOTINFO.DAT` 中指定的路径放置
