# 数据格式

MHVN98 运行时读取的数据文件由工具链（MHVNSCAS、MHVNTXAR、MHVNIMGP、MHVNLINK）生成。以下仅列出引擎运行时实际解析的格式。

## ROOTINFO.DAT

引擎唯一硬编码路径的文件，必须在当前目录。

**加载**: `src/rootinfo.c:ReadInRootInfo()` (行 35–78)

```c
0x00  char[4]   magic           "MHVN"
0x04  uint16    VNflags         高级行为标志
0x06  uint16    numstvar_glob   全局变量数
0x08  uint16    numflags_glob   全局标志数
0x0A  uint16    numstvar_loc    存档局部变量数
0x0C  uint16    numflags_loc    存档局部标志数
0x0E  uint16    format_norm     默认文本格式
0x10  uint16    format_char     角色名文本格式
0x12  uint16    format_menu_notsel 未选中菜单项格式
0x14  uint16    format_menu_sel    选中菜单项格式
0x16  char[12]  font_file       字体文件名
0x22  char[12]  scenedat_file   场景数据文件名
0x2E  char[12]  langdat_file    语言数据文件名
0x3A  char[12]  bgdat_file      背景图像数据文件名
0x46  char[12]  sprdat_file     精灵数据文件名
0x52  char[12]  musdat_file     音乐数据文件名
0x5E  char[12]  sedat_file      音效数据文件名
0x6A  char[12]  systemdat_file  系统数据文件名
```

### VNflags

```c
bit 0 (C) — 展示 CG Gallery（0=隐藏）
bit 1 (M) — 展示 Music Room（0=隐藏）
bit 2 (P) — 自定义主角支持
```

## 场景数据文件 (SCENE.DAT)

**加载**: `src/sceneengine.c:SetupSceneEngine()` / `LoadNewScene()`

```
0x00  uint16          num_scenes    总场景数
0x02  uint16          num_chars     总角色数
0x04  uint32[num_scenes] sceneptrs  每个场景数据的指针
var   LZ4_data[]                第 0~num_scenes-1 个场景的压缩数据
```

每个场景解压后（最大 4096 字节）包含场景字节码（详见 [07-场景字节码指令集.md](07-场景字节码指令集.md)）。

## 语言数据文件 (LANGUAGE.DAT)

**加载**: `src/rootinfo.c:InitLanguage()` (行 80–102)

```
0x00  uint16            num_lang         支持的语言数
0x02  uint32[num_lang]  textdat_file_ptrs 每种语言的 TXA 文件路径偏移
var   char[][]          textdat_files     TXA 文件名数组
```

`InitLanguage(0)` 读取第 0 种语言的 TXA 文件路径到 `rootInfo.curTextDataPath`。

## 文本数据文件 (*.TXA)

**加载**: `src/textengine.c:SetupTextInfo()` / `LoadSceneText()` / `LoadCurrentCharacterName()`

```
0x00  uint32  systext_ptr     系统文本指针
0x04  uint32  credits_ptr     致谢文本指针
0x08  uint32  charnames_ptr   角色名指针
0x0C  uint32  scenetext_ptr   场景文本指针
0x10  uint32  cgtext_ptr      CG 名称指针（可为 0）
0x14  uint32  musictext_ptr   音乐名称指针（可为 0）
```

### 系统文本 (按顺序)

```
LANGNAME           语言原生名（如 "日本語"）
SYS_MENU_NEWSTORY  "Start New"
SYS_MENU_CONTINUE  "Continue"
SYS_MENU_CONFIG    "Options"
SYS_MENU_QUIT      "Quit"
SYS_MENU_GALLERY   "CG Gallery"
SYS_MENU_MUSICROOM "Music Room"
```

### 角色名

```
uint16[num_chars]  charnames_ptrs  每个角色名的偏移
char[]             charnames_text  角色名字符串
```

### 场景文本

```
uint32[num_scenes] scenetext_ptrs  每个场景文本块的偏移
对于每个场景:
  uint16          num_strings   字符串数
  uint16[num_strings] text_ptrs 字符串偏移
  char[]          text_array    LZ4 压缩的文本数据
```

## 背景图像数据文件 (BGIMAGE.DAT)

**加载**: `src/graphics.c:LoadBGImage()`

```
0x00  uint32               bgmain_ptr      背景指针表偏移
0x04  uint16               num_bg          背景数量
0x06  uint16               num_pal         调色板数量
0x08  uint4[3][16][num_pal] palette_table  调色板数据
var   uint32[num_bg]       bg_table        每个背景信息的偏移

每个背景:
  0x00  uint32         mainbg_ptr    主图像数据偏移
  0x04  uint16         num_variant   变体数
  0x06  uint16         palette_index 调色板索引
  0x08  variant_data[num_variant]    变体数据

每个变体:
  0x00  uint32  variant_ptr  变体图像偏移
  0x04  int16   xpos         X 位置
  0x06  int16   ypos         Y 位置
```

图像数据为无调色板头部的 GPI 文件格式。

## 精灵数据文件 (SPRITE.DAT)

**加载**: 桩（当前未使用，`setspr0-2` 指令为空）

```
0x00  uint16           num_spr      精灵数量
0x02  uint32[num_spr]  spr_table    每个精灵信息的偏移

每个精灵:
  0x00  uint32         mainspr_ptr  主图像偏移
  0x04  uint16         num_variant  变体数
  0x06  variant_data[num_variant]   变体数据
```

变体格式与背景相同。

## 字体文件

详见 [09-Unicode-与字体系统.md](09-Unicode-与字体系统.md)。

## 音乐/音效/系统数据文件

当前均为未使用的占位符。引擎中对应的路径从 `ROOTINFO.DAT` 读取，但无实际加载逻辑。
