# 中断处理

## VSync 中断服务例程

**源文件**: `src/isr.asm` (43 行), `src/isr.h` (31 行)

### 实现

```asm
VsyncInterrupt:
    push ax
    push ds

    ; 设置 DS = SS（数据段 = 堆栈段）
    mov ax, ss
    mov ds, ax

    ; 设置 VSync 标志
    inc byte ptr [vsynced]

    ; 复位 GDC 中断
    mov dx, 0x60
    mov ax, 0x0A
    out dx, ax

    ; 向主 PIC 发送 EOI
    mov al, 0x20
    out 0x00, al

    pop ds
    pop ax
    iret
```

### `vsynced` 标志

```c
// isr.h
extern unsigned char vsynced;  // 在 isr.asm 的 .bss 段声明
```

主循环中使用：
```c
while (!vsynced) { __asm volatile ("hlt"); }  // 等待 VSync
// ... 帧处理 ...
vsynced = 0;  // 清理标志
```

## 中断控制器管理

**源文件**: `src/platform/pc98_interrupt.h`

PC-98 使用两个级联的 8259A 可编程中断控制器 (PIC)：

### 主 PIC (8259A #1)

| 端口 | 用途 |
|------|------|
| `0x00` | IMR (中断掩码寄存器) / OCW1 |
| `0x02` | OCW2 / OCW3 |

### 从 PIC (8259A #2)

| 端口 | 用途 |
|------|------|
| `0x04` | IMR |
| `0x06` | OCW2 |

### 相关中断

```c
#define INTERRUPT_MASK_VSYNC    0x20  // 位 5 — CRT VSync
#define INTERRUPT_MASK_TIMER    0x01  // 位 0 — 定时器
#define INTERRUPT_MASK_KEYBOARD 0x02  // 位 1 — 键盘

#define INTERRUPT_VECTOR_VSYNC   0x08  // 中断向量号 8
#define INTERRUPT_VECTOR_TIMER   0x1C  // 中断向量号 0x1C
#define INTERRUPT_VECTOR_KEYBOARD 0x09 // 中断向量号 9
```

## 中断管理函数

### 掩码控制

```c
unsigned char PC98GetPrimaryInterruptMask(void);
// 读取主 PIC 的 IMR

void PC98SetPrimaryInterruptMask(unsigned char mask);
// 写入主 PIC 的 IMR

void PC98AddPrimaryInterrupts(unsigned char mask);
// 启用特定中断: IMR &= ~mask (0 = 启用)

void PC98RemovePrimaryInterrupts(unsigned char mask);
// 禁用特定中断: IMR |= mask (1 = 禁用)
```

### EOI (中断结束)

```c
void PC98PrimaryEndOfInterrupt(void);
// OUT 0x00, 0x20 — 非指定 EOI 命令
```

### 中断向量操作

```c
// 读取中断向量表 (IVT) 中的函数指针
InterruptFuncPtr GetInterruptFunctionRaw(unsigned char vec);

// 设置中断向量表 (IVT) 中的函数指针
void SetInterruptFunctionRaw(unsigned char vec, InterruptFuncPtr func);
```

## 引擎中的中断流程

### 初始化

```c
oldInterruptMask = PC98GetPrimaryInterruptMask();     // 保存原掩码
intsoff();                                            // CLI
oldVsyncVector = GetInterruptFunctionRaw(VSYNC);      // 保存原向量
SetInterruptFunctionRaw(VSYNC, VsyncInterrupt);       // 安装新向量
PC98AddPrimaryInterrupts(INTERRUPT_MASK_VSYNC);       // 启用 VSync 中断
intson();                                             // STI
```

### 清理

```c
intsoff();
PC98SetPrimaryInterruptMask(oldInterruptMask);         // 恢复原掩码
SetInterruptFunctionRaw(VSYNC, oldVsyncVector);        // 恢复原向量
intson();
```

## VSync 时序

PC-98 的 VSync 中断频率取决于显示模式。在 640×400 模式下，约为 **56.4 Hz**。这意味着主循环每秒钟执行约 56.4 帧，适合文字逐字动画的流畅显示。

## 中断安全

引擎在关键操作前后使用 `intsoff()`/`intson()`（即 `CLI`/`STI`）来确保原子性。VSync 中断服务例程非常短小（仅设置标志、复位 GDC、发送 EOI），最大程度减少中断延迟。
