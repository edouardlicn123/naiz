# 实用工具函数

## LZ4 解压缩

MHVN98 集成了 Trixter/Hornet 优化的 8086 汇编实现的 LZ4 解压缩器。该实现用于解压场景数据和文本数据。

### 汇编实现

**源文件**: `src/lz48086.asm` (321 行)

这是一个为实模式 8086 优化的 LZ4 解压缩器：

- 从 `[SI+4]` 读取压缩数据，解压到 `[DI]`
- 在 `[SI]` 处存储的 4 字节是**压缩块的总大小**（包含这 4 字节本身）
- 支持 LZ4 的所有标准指令：字面量复制、偏移匹配、短匹配、RLE 填充
- 包含 `SHR4table` 查找表用于 `shr 4` 操作加速
- 提供两个版本：
  - `lz4_decompress` — 速度优化版本
  - `lz4_decompress_small` — 体积优化版本（被注释掉）

### C 包装

**源文件**: `src/lz4.c` (29 行)

```c
unsigned int LZ4Decompress(unsigned char* dst, unsigned char* src);
```

C 包装器处理段寄存器切换，调用汇编函数，然后恢复段寄存器。

### 使用方法

```c
// 压缩数据格式:
// [0x00] uint32  compressed_size  // 包含此 4 字节头
// [0x04] uint8[] compressed_data

*((__far unsigned long*)decompbuf) = compSize;  // 设置压缩大小
LZ4Decompress(targetBuffer, decompbuf);          // 解压
```

## 三角函数

**源文件**: `src/stdbuffer.c` (226 行)

所有三角函数使用全定点数运算，基于预计算查找表。

### `Sin(x)` (行 82–87)

```c
short Sin(unsigned int x);
// x: 1.15 定点数, 0x0000-0xFFFF 映射到 0-2π
// 返回值: 2.14 定点数 (-16384 ~ +16384)
```

- 使用 256 条目的半正弦表 (`sineTable[256]`)
- 通过 `x>>8` 索引，`x>>6` 确定象限
- 第二半周期取反

### `Cos(x)` (行 89–93)

```c
short Cos(unsigned int x);
// 实现: return Sin(x + 0x2000);  // 相位偏移 90°
```

### `Atan2(y, x)` (行 95–194)

```c
unsigned int Atan2(short y, short x);
// 返回值: 1.15 定点数, 0-0xFFFF 映射到 0-2π
```

- 使用 16×16 的反正切表 (`atan2Table[256]`)
- 处理象限对称性
- 处理 y=0 和 x=0 的轴边界情况
- MSB 归一化以选择最精确的表条目
- 仅在输入精度损失较大时进行细化

### `Sqrt(x)` (行 197–225)

```c
unsigned short Sqrt(unsigned long x);
```

- 使用 256 条目的整数平方根表 (`sqrtTable[256]`) 做初始估计
- 牛顿迭代法收敛
- 迭代过程：`x1 = (x0 + x/x0) / 2`，直到 `x1 >= x0`

## 查找表

### 正弦表 (256 条目, 2.14 定点)

覆盖半周期 (0-π)，值范围 `[0, 16384]`。全周期通过符号反转实现。

### 反正切表 (256 条目, 1.15 定点)

覆盖右上象限 (x>0, y>0)，值范围 `[0, 4096)`。其他象限通过对称性变换。

### 平方根表 (256 条目)

```c
const unsigned char sqrtTable[256] = { 0,  1,  1,  1,  2,  2, ... };
```

输入 `x` 的平方根的值，用于牛顿法的初始估值。

## 小文件缓冲区

```c
unsigned char smallFileBuffer[1024];
```

全局 1KB 缓冲区，用于各种临时文件读取操作（如读取 ROOTINFO.DAT 头、文件指针表等）。设计为函数间的临时共享缓冲区。
