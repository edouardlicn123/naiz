# MHVN98 概述

## 项目定位

MHVN98 是 **MHVN Visual Novel Engine** 的 PC-98 运行时引擎。它是整个引擎管线中的最终执行端，负责在 PC-98 实模式 DOS 环境下加载、解释并渲染视觉小说。

- **作者**: Maxim Hoxha
- **协议**: MIT
- **目标平台**: PC-98 系列（需要 EGC 硬件支持）
- **编译平台**: Linux / Windows（通过 MinGW-w64 交叉编译）
- **工具链**: `ia16-elf-gcc` (gcc-ia16)，`-march=i386 -mcmodel=small`

## 架构总览

```
┌─────────────────────────────────────────────────────────────┐
│                       主循环 (main.c)                        │
│  vsync 等待 → 绘制请求处理 → 场景字节码处理 → 异步动作更新  │
│  → 键盘输入处理 → 下一帧                                    │
└──────────┬──────────────────────────────────────┬───────────┘
           │                                      │
     ┌─────▼──────┐                        ┌──────▼──────┐
     │ 场景引擎    │    ┌──────────────┐    │ 文本引擎    │
     │ sceneengine │◄───│ ROOTINFO.DAT │───►│ textengine  │
     └─────┬──────┘    └──────────────┘    └──────┬──────┘
           │                                      │
     ┌─────▼──────┐                        ┌──────▼──────┐
     │ 图形系统    │                        │ Unicode/字体 │
     │ graphics    │◄──► 调色板 palette    │ unicode     │
     └─────┬──────┘                        └─────────────┘
           │
     ┌─────▼─────────────────────────────────────────────┐
     │              平台抽象层 (platform/)                 │
     │  EGC · GRCG · GDC · 键盘 · 文件I/O · 内存 · 端口   │
     └───────────────────────────────────────────────────┘
```
